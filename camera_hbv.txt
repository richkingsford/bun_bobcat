Camera: HBV HD CAMERA (generic USB webcam)
USB vendor:   0ac8:0346  (Z-Star Microelectronics)
Bus info:     usb-xhci-hcd.2.auto-1.3
Device nodes: /dev/video2  (capture),  /dev/video3  (metadata)
Protocol:     V4L2 / UVC  (standard — works with cv2, ffmpeg, fswebcam)

Format:
  Resolution:   1280x720
  Pixel format: MJPG (Motion-JPEG)

Python (opencv):
  import cv2
  cap = cv2.VideoCapture('/dev/video2', cv2.CAP_V4L2)
  cap.set(cv2.CAP_PROP_FRAME_WIDTH, 1280)
  cap.set(cv2.CAP_PROP_FRAME_HEIGHT, 720)
  ret, frame = cap.read()
  cap.release()
  if ret:
      cv2.imwrite("photo.jpg", frame)

Terminal (no Python):
  fswebcam -d /dev/video2 --no-banner -r 1280x720 photo.jpg

No udev rules or special permissions needed — accessible as standard UVC device.
